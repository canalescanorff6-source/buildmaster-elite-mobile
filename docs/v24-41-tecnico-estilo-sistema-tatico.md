# v24.41 — Técnico, Estilo e Sistema Tático

- Cadastro organizado por versão do técnico.
- Estilo principal e secundário com proficiência.
- Grupos Lendário/Épico, Pacotes/Seleções e GP.
- Selecionar um técnico aplica seu estilo principal, mas o usuário pode trocar para o secundário.
- A proficiência refina a avaliação tática e as simulações de ficha.
- A posição escolhida pelo usuário continua soberana.
- O técnico nunca bloqueia, troca ou redefine a posição.
- Dados iniciais marcados como informados pelo usuário para permitir atualização futura por pacote/temporada.
