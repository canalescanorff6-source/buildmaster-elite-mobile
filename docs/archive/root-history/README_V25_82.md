# BuildMaster Elite Tático v25.82 — Fichas Únicas por Carta

## Correção principal

O motor deixou de usar a posição como molde dominante. A posição escolhida continua definindo as exigências mínimas, mas cada ficha agora é diferenciada por:

- atributos reais e diferenças contínuas entre as cartas;
- Estilo de Jogo oficial;
- habilidades oficiais e especiais;
- altura e peso confirmados;
- pontos fortes naturais que devem ser preservados;
- fraquezas realmente necessárias para a posição escolhida;
- regras locais de jogadores conhecidos, quando existentes.

O resultado inclui uma assinatura individual e explica por que a ficha é diferente de outras cartas na mesma posição. O perfil técnico exibido é apenas uma descrição do app e nunca substitui o Estilo de Jogo oficial.
