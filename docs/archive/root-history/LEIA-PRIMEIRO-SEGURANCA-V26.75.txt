BUILDMASTER ELITE TÁTICO v26.75 — SEGURANÇA REFORÇADA

ATENÇÃO: siga esta ordem para não bloquear seu próprio acesso.

1. Primeiro envie este projeto ao GitHub e gere o APK v26.75.
2. Baixe o APK v26.75 e deixe o arquivo guardado no celular/computador.
3. Só depois publique a segurança no Supabase:
   - execute 1-INSTALAR-SEGURANCA-V26.75.sql no SQL Editor;
   - substitua a função license-session pelo arquivo 2-license-session.txt;
   - substitua a função admin-users pelo arquivo 3-admin-users.txt.
4. Instale o APK v26.75 e entre novamente.
5. Abra Ajustes > Contas > Segurança do administrador e ative o MFA.

Depois da etapa 3, APKs anteriores à v26.75 serão bloqueados pelo servidor.
Os vínculos antigos de aparelhos serão desconectados uma vez; isso é esperado.
Não apague usuários, banco ou Cofres.
