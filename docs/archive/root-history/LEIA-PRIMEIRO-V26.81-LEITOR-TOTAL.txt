BUILDMASTER ELITE TÁTICO v26.81 — LEITOR TOTAL DE CARTA

PACOTE COMPLETO
Use este projeto como substituto da v26.80.

COMO LER UMA CARTA COM MAIS PRECISÃO
1. Abra Criar Ficha.
2. Escolha “Leitura completa”.
3. Envie obrigatoriamente:
   - Visão geral;
   - Atributos;
   - Progressão e pontos.
4. Envie Habilidades para aumentar a precisão.
5. Envie Posições jogáveis quando disponível.
6. Toque em “Analisar carta completa”.
7. Confirme apenas os campos destacados na revisão.

PROTEÇÕES
- O app compara os prints para evitar misturar cartas diferentes.
- A ficha final fica bloqueada quando faltam dados críticos.
- Se houver divergência, será necessário confirmar que todos os prints pertencem à mesma versão da carta.
- A posição escolhida pelo usuário continua soberana.
