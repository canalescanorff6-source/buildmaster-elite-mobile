# BuildMaster Elite Tático v24.4 — Habilidades Oficiais Stable

Correção focada em goleiros, mantendo a base estável da v24.

## O que mudou

- Motor separado para GOL/GK.
- Habilidades de jogador de linha não aparecem mais como prioridade para goleiro.
- Ficha de goleiro zera Finalização, Passe, Drible e Defesa quando a posição final é GOL.
- Plano usa Goleiro 1, Goleiro 2, Goleiro 3, Bola aérea e Força nas pernas conforme o perfil.
- Objetivo novo: Goleiro elite.
- Variantes novas: ficha goleiro segura, ficha goleiro reflexo e ficha goleiro reposição.
- Explicação específica usando Talento de GO, Firmeza, Defesa, Reflexos, Alcance, Salto e Contato físico.

## Teste incluído

Foi adicionado teste de regressão com goleiro defensivo para garantir que o app não recomenda Interceptação, Bloqueador, Marcação individual, Chute de primeira, Passe de primeira ou Toque duplo como prioridade de goleiro.
