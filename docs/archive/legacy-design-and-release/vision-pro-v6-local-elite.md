# BuildMaster Local Pro v6

Esta versão trabalha sem API paga. A leitura é feita no navegador com OCR local e o motor de ficha usa regras próprias para desempenho real no eFootball.

## Regras principais

- Ler ficha automática/treino distribuído quando aparecer no print.
- Somar os pontos pelo custo real progressivo do eFootball.
- Redistribuir os pontos com motor Elite Local.
- Não recomendar posição fora da carta lida.
- Não aceitar pontos absurdos como 2/2 ou 116/116.
- Não repetir habilidades que o jogador já possui.

## Fluxo recomendado

1. Abra a carta do jogador com atributos e ficha automática visível.
2. Tire print direto da tela.
3. Envie no app pelo botão Buscar print/galeria.
4. O app lê localmente e gera a ficha.
