# Card visual com imagem do jogador

Esta versão coloca a imagem enviada pelo usuário dentro do card premium do resultado.

Como funciona:

1. O usuário envia o print da carta.
2. O app faz um recorte aproximado da área do card do jogador em prints do eFHUB/eFootBase.
3. Esse recorte é usado no card visual premium do BuildMaster AI.
4. O overall, posição PT-BR, estilo, PRI, build e habilidades continuam sendo calculados pela análise.

Observação: o recorte é automático e aproximado. Para melhor resultado, use print direto e nítido da ficha da carta, sem cortar as bordas.
