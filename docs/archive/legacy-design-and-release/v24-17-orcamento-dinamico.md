# BuildMaster Elite Tático v24.17 — Orçamento Dinâmico

Correção principal: a ficha agora respeita exatamente os pontos/nível digitados pelo usuário na Auditoria Elite.

## Prioridade de orçamento

1. Pontos de progresso disponíveis digitados manualmente.
2. Nível máximo digitado manualmente, convertido para pontos.
3. Ficha automática lida do print.
4. Nível lido pelo OCR.
5. Pontos lidos pelo OCR.
6. Padrão seguro.

Isso impede que o print antigo ou a ficha automática sobreponha a quantidade de pontos que o usuário informou.

## Exemplo

Se o usuário digitar 62 pontos, todas as fichas segura, competitiva e alternativa são encaixadas dentro de 62 pontos pelo custo progressivo real do eFootball.
