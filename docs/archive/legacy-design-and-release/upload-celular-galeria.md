# Upload no celular: galeria ou câmera

Esta versão separa o envio de imagem em dois botões:

- **Buscar print/galeria**: abre o seletor de arquivos/galeria do celular, ideal para selecionar prints do eFHUB/eFootBase.
- **Tirar foto**: abre diretamente a câmera do celular.

Antes havia apenas um input com `capture="environment"`, e muitos navegadores Android abriam a câmera automaticamente. Agora o usuário escolhe o fluxo correto.

Para maior precisão no OCR, o recomendado é usar print direto da tela, não foto de outro aparelho.
