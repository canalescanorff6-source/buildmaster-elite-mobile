# BuildMaster Elite Tático v24.24 — Design Premium completo

Esta versão mantém a base fiel do projeto e melhora a experiência visual do app.

## Principais mudanças

- Dashboard inicial com visual premium.
- Central de comando com atalhos grandes para Leitor Elite, Manual Pro, Cofre e Guia rápido.
- Selos visuais de precisão: pontos exatos, lista oficial e visual premium.
- Barra de status do fluxo: Entrada, Auditoria, Plano final e Salvo.
- Melhor leitura no celular com botões maiores, cards mais compactos e ações fixas.
- Abas do resultado com comportamento sticky e ações finais sempre acessíveis.
- Tema claro/escuro e cores de destaque mais consistentes.
- Cache/PWA atualizado para v24.24.

## O que não foi alterado

- Motor de ficha.
- Regras de pontos.
- Habilidades oficiais.
- Goleiro.
- Mapa total do time.
- Cofre e sincronização.
