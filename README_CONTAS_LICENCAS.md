# Contas e licenças

O administrador cria, renova, bloqueia e gerencia contas pelo painel protegido. A sessão, prazo, aparelho autorizado e versão mínima são validados no servidor.

Ações administrativas exigem a segurança configurada na base v26.75, incluindo MFA quando aplicável. Não habilite cadastro público de usuários no Supabase.
