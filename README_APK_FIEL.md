# APK Fiel — BuildMaster Elite Tático v24.29

Esta versão mantém o visual fiel do seu projeto e adiciona a etapa **Regras Atualizáveis**.

## Como gerar

1. Suba os arquivos no GitHub.
2. Vá em **Actions**.
3. Rode:

```text
Gerar APK Android Fiel v24.29 Regras Atualizáveis
```

4. Baixe o artifact do APK.

## Observação

As correções locais ficam no armazenamento do app/navegador. Se você desinstalar o APK ou limpar os dados do app, pode perder essas correções locais.
