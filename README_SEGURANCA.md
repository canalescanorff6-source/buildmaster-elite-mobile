# Segurança vigente

A v27.10 preserva a base de segurança v26.75:

- versão mínima validada no servidor;
- MFA administrativo;
- tokens protegidos pelo Android Keystore;
- APK, URL, SHA-256, pacote e assinatura verificados;
- janela offline limitada;
- backup criptografado;
- controle assinado de aparelho;
- rate limit administrativo.

A v27.10 acrescenta diagnóstico com remoção de segredos e regras de `.gitignore` para impedir o envio acidental de arquivos de assinatura e ambiente.
