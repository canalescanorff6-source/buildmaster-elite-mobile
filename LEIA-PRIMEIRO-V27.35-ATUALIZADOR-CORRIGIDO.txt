BUILDMASTER ELITE TÁTICO v27.35 — LEIA PRIMEIRO

CAUSA REAL:
O plugin Android usava PluginCall.getLong() para ler o versionCode. O Capacitor entregava esse número como Integer, então o plugin recebia null e mostrava “URL, versão ou SHA-256 inválido” antes mesmo de baixar o APK.

CORREÇÃO:
A v27.35 converte qualquer Number para long e mantém DownloadManager + HTTP reserva, SHA-256, pacote, versionCode e assinatura.

AVISO DE ARMAZENAMENTO:
A cópia pesada e duplicada do Cofre no localStorage foi removida. O histórico integral permanece no IndexedDB; o fallback local agora é compacto.

IMPORTANTE:
A versão atualmente instalada possui o plugin nativo antigo. Ela não consegue baixar automaticamente a correção do próprio plugin. Instale a v27.35 manualmente UMA ÚNICA VEZ, por cima do app atual, sem desinstalar. Depois publique uma v27.36 para comprovar a atualização automática.

NÃO ALTERE:
- Supabase
- ANDROID_SIGNING_BUNDLE
- applicationId

PUBLICAÇÃO:
1. Substitua todo o repositório, inclusive .github.
2. Faça commit na main.
3. Aguarde o workflow terminar verde.
4. Baixe e instale o APK v27.35 uma única vez.
5. Publique a próxima versão e teste pelo botão Atualizar.
