# APK nativo — BuildMaster v27.10

O projeto usa Capacitor para executar a interface embarcada no APK e plugins nativos para operações Android, incluindo atualização e armazenamento seguro.

O workflow executa instalação travada, TypeScript, testes, exportação estática, sincronização Android, compilação, assinatura, inspeção do APK e publicação da release.

Alterações no código exigem um novo APK. O Android sempre pede confirmação do usuário para instalar uma atualização distribuída fora da Play Store.
